# 03. Longest Substring Without Repeating Characters

### Solution-1 O\(n\)

```java
public class Solution {
    public int lengthOfLongestSubstring(String s) {
        HashSet<Character> set = new HashSet<>();
        int n = s.length();
        int i = 0,j = 0,max = 0;
        
        while(i < n && j < n){
            if(!set.contains(s.charAt(j))){
                set.add(s.charAt(j++));
                max = Math.max(max,j-i);//jはすでに+1だからそのままでいい
                //--------------------------------
                // for(Object obj : set)
                //     System.out.print(obj);
                // System.out.println();
                //--------------------------------
            }
            else
                set.remove(s.charAt(i++));
        }
        return max;
    }
}
/*思路
  从字符串的开头开始遍历，用hashset判断第一个字母到第二个字母所构成的
  sub字符串是否重复，然后第一个到第三个，以此类推。若第一个到第n个出现
  重复，删除hashset里的字符串的第一个字母，相当于判断第二个字母到第n个
  字母是否出现重复，以此类推直到字符串结尾。
*/
/*
  优点，只需要一次遍历就可以得到结果，减去solution2中的重复动作，
  时间和空间复杂度上都好很多
*/
```

### Solution-2 O\(n^3\)

```java
public class Solution {
    public int lengthOfLongestSubstring(String s) {
        int max = 0; int n = s.length();
        for(int i = 0; i < n; i++){
            for(int j = i+1; j <= n; j++){
                if(Unique(i,j,s)){
                    max = Math.max(j-i,max);
                }
            }
        }
        return max;
    }
    public boolean Unique(int start,int end,String s){
        HashSet<Character> set = new HashSet<>();
        
        for(int i = start; i < end; i++){
            if(set.contains(s.charAt(i))){
                System.out.print("!");
                return false;
            }
                set.add(s.charAt(i));
        }
        Iterator<Character> ir = set.iterator();
        while(ir.hasNext()){
            System.out.print(ir.next() + " ");
        }
        System.out.println();
        return true;
    }
}
/*思路
  从字符串的开头开始遍历，判断第一个字母到第二个字母所构成的
  sub字符串是否重复，第一个到第三个，以此类推。这是一个大循环。
  然后判断从第二个字母开始到第三个字母是否重复，以此类推。
  直到所有循环结束，遍历了字符串从任意一个字母开始到最后一个字母
  所构成的sub字符串是否包含重复字符。
  期间不断更新不重复的sub字符串的开头索引i及末尾索引j的差到结果
*/
/*
  缺点，时间及空间复杂度过高，每次判断重复都需要新建一个哈希set
*/
```
