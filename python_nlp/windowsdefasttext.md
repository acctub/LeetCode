# WindowsでfastText

STEP１各種ライブラリをインストール

```bash
$ wget https://github.com/facebookresearch/fastText/archive/v0.1.0.zip
$ unzip v0.1.0.zip
$ cd fastText-0.1.0
$ make
```

fasttextのpython用のライブラリのインストール

```bash
$ git clone https://github.com/facebookresearch/fastText.git
$ cd fastText
$ pip install .
```
